# 💀 O que sobra pro beta?

Um site simples e divertido feito com **HTML, CSS e JavaScript**.

## 📋 Descrição

O usuário recebe uma pergunta extremamente importante:

> "O que sobra pro beta?"

Com duas opções:

- ✅ Tudo
- 💀 Nada

Ao clicar em **Nada**, uma imagem aparece com animações, efeitos visuais, partículas e um overlay estilizado.

## ✨ Recursos

- Interface responsiva
- Efeito glitch no título
- Partículas animadas no fundo
- Botões com animações e efeitos de brilho
- Overlay animado
- Exibição de imagem ao selecionar a resposta
- Desenvolvido apenas com HTML, CSS e JavaScript

## 🚀 Como usar

1. Baixe o arquivo `index.html`
2. Abra no navegador
3. Escolha uma das opções
4. Descubra o destino do beta 💀

## 📁 Estrutura

```text

 ├── index.html
 └── README.md
```
